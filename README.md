## Описание программы

Movie Library — это графическое приложение для ведения личной коллекции фильмов. Программа позволяет добавлять информацию о фильмах (название, жанр, год выпуска, рейтинг), просматривать их в виде таблицы, фильтровать по жанру и году, а также сохранять данные в JSON-файл для последующей загрузки.

---

## Возможности программы

- Добавление фильмов с проверкой корректности ввода (год от 1888 до 2030, рейтинг от 0 до 10)
- Просмотр всех добавленных фильмов в таблице
- Фильтрация по жанру (частичное совпадение)
- Фильтрация по году (точное совпадение)
- Сброс фильтров для возврата к полному списку
- Автоматическое сохранение данных в файл `movies.json`
- Автоматическая загрузка данных из `movies.json` при запуске
- При первом запуске создаётся демо-набор из 10 фильмов
